#JOBEEE API
>Backend RESTFUL API for jobs build in Node.js using Express.js & MongoDB

## Install all Dependencies
```
npm i
```
## Config.env
Create a `.env` file in the project root with the following variables:

```
PORT=
MONGODB_URI=
NODE_ENV=
COOKIE_EXPIRES_TIME=
JWT_SECRET=''
JWT_EXPIRES_TIME=
SMTP_HOST=
SMTP_PORT=
SMTP_USER=
SMTP_PASS=
```

Replace each value with your own configuration.
